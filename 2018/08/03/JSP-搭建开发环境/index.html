JSP简介

JSP(Java Server Pages)是在传统的网页HTML文件中插入Java程序段和JSP标记，从而形成JSP文件,后缀名为(*.jsp)。与PHP和ASP.net是目前主流的Web应用程序开发技术路线。与两者相比较具有高性能，跨平台的优势,大型电子商务站点大多使用JSP开发，淘宝网，百度等都大量使用JSP技术。这篇博客着重介绍开发环境的搭建,Linux和Windows环境下都介绍;Intellj和Eclipse也都用。以及介绍JSP项目文件。

Tomcat

如IIS之于ASP,JSP也要配置好服务器，这里选择Tomcat。Tomcat 官网下载可能有点慢，推荐访问清华大学开源软件镜像站，这个网址 https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-9/v9.0.10/bin/

创建步骤

File –> New –> Dynamic Web project

  • 1.填写项目名称
  • 2.绑定之前下载并配置好的tomcat服务器(路径为之前安装的路径)
  • 3.选择自己的JDK版本

创建在WebContent 文件夹下创建JSP文件,添加测试代码,并运行，在浏览器输入localhsot:8080/w项目名/Jsp文件名访问

项目文件分析

  • Deployment Descripton : 网站部署的描述
  • Web Libraries : 自己加的包可以放在里面
  • build : 存放编译之后的文件
  • WebCotent :存放写入的jsp页面
  • WEB-INF文件夹下面的内容，是安全目录，客户端是无法访问的，还有web.xml是项目部署文件，可以通过test.jsp设置网站欢迎页面
