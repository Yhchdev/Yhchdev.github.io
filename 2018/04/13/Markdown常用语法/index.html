那么问题来了如何在hexo中添加本地图片呢？

  1. 修改配置文件 _config.yml 里的 post_asset_folder 的属性值为 true;
  2. 要将本地图片上传至github需要装一个插件;
npm install hexo-asset-image --save

3.当创建文章的时候（hexo new “text”)，会发现 /soure/_post 下除了 test.md 会多了一个 test的同名文件夹;
4.这时我们将要引入的图片，丢进这个同名文件夹即可，编辑博客的时候采用markdown语法插入图片即可

小结

markdown并不是针对程序员设计的，它的目的是让人写文档时，更加专注于文档的内容，而不用花很多时间去美化格式，事实上当你用熟它的时候，你会发现，当你写完一篇文章时，就结束了(当然你需要检查一遍)，一篇格式优美的文档就呈现在你面前了。
